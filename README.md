### Solving Steps
1. Record start time
2. Read the problem
3. Think about questions (Don't look at the sample)   
  ex) Input data type, limit time-complexity ..
4. Find idea 
5. Write pseudocode
6. Coding
7. Submit code
8. Record end time if succeed
### Target time to find idea
* Easy : 1-4 minutes
* Medium : 1-8 minutes
* Hard : 1-12 minutes
